Modern, professional blog platform with a clean, minimalist aesthetic. The design should focus on readability and user engagement with a cohesive visual language across all pages.

**DESIGN SYSTEM (REQUIRED):**
- Platform: Web, Desktop-first responsive design
- Palette: 
  - Primary: Indigo (#6366f1) for key actions and links
  - Secondary: Teal (#14b8a6) for accents and secondary actions
  - Surface: Light gray (#f8fafc) for cards and containers
  - Background: White (#ffffff) for main backgrounds
  - Text: Dark gray (#334155) for primary content
- Styles: Subtle rounded corners (8px), soft shadows for depth, generous whitespace

**PAGE STRUCTURE:**
1. **Header:** Sticky top navigation with logo on left, search bar in center, and user controls on right. Mobile version with hamburger menu and bottom navigation bar.
2. **Hero Section:** Featured post card with large image, category tags, author information, and engagement metrics.
3. **Primary Content Area:** 
   - Grid layout for blog post cards with featured image, title, excerpt, author info, and interaction buttons
   - Filter tabs for sorting (Relevant, Latest, Top)
   - Category sidebar with tag cloud
   - Right sidebar with trending posts and newsletter signup
4. **Footer:** Multi-column layout with site navigation, social links, and copyright information

**REQUIRED PAGES:**
1. **Homepage** - Featured post, post listing with sorting options, category sidebar
2. **Individual Post Page** - Full post content with featured image, author bio, comment section, like/bookmark buttons
3. **Create Post Page** - Form with title input, rich text editor, image upload, category selection
4. **User Dashboard** - User profile, post management (edit/delete), analytics overview
5. **Category Page** - Posts filtered by category with related categories sidebar
6. **Search Results Page** - Search functionality with filters and result listing
7. **Bookmarks Page** - User's saved posts with organization options
8. **Admin Panel** - User management, post moderation, report handling
9. **User Profile Page** - User information, authored posts, activity feed
10. **Authentication Pages** - Login and registration forms with social login options

**INTERACTIVE COMPONENTS:**
- Like/Bookmark buttons with real-time counters
- Comment system with nested replies
- Dark/light mode toggle
- Responsive navigation that adapts to mobile
- Search with autocomplete suggestions
- Rich text editor for content creation
- Image upload with preview
- Notification system for user interactions
